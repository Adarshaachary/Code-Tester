# 💻 Code-Tester

Code-Tester is a frontend web application that brings together useful coding resources in one place. Instead of opening multiple websites while learning or practicing programming, this project provides quick access to coding platforms, interview preparation resources, developer tools, and other helpful links through a simple and responsive interface.

The goal of this project is to make coding resources easier to access while also practicing frontend development concepts such as responsive layouts, JavaScript interactivity, and modern UI design.

## 🚀 Live Demo

🔗 https://adarshaachary.github.io/Code-Tester/web.html

## Features

- Responsive design that works across desktop, tablet, and mobile devices.
- Clean and easy-to-use interface.
- Dark mode for a better viewing experience.
- Smooth navigation between different sections.
- Interactive features built using JavaScript.
- Quick access to popular coding platforms and developer resources.
- Organized layout for learning, practice, and interview preparation.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5

## Why I Built This

I created Code-Tester to have a single place where I could quickly access the websites and tools I use while learning programming. At the same time, it helped me improve my frontend development skills by building a responsive and interactive web application from scratch.
